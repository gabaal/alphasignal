# AlphaSignal: Institutional Alpha Terminal (V2.0)

AlphaSignal is a professional intelligence terminal and risk management platform for digital asset portfolios. Powered by an on-chain ML Alpha Engine, it provides real-time narrative synthesis, institutional order flow deep-dives, and sophisticated portfolio simulation.

## 🚀 Quick Start
1.  **Environment**: Ensure dependencies are installed via `pip install -r requirements.txt`. 
2.  **Launch**: Run `python main.py`.
3.  **Access**: Navigate to `http://localhost:8006` in your browser.

## 🏛️ Core Modules

### 🛰️ Institutional Intelligence Briefing (Overview)
Live briefing engine that synthesizes on-chain momentum, ML alpha predictions, and emerging narratives into a tactical "Intelligence Stream."

### 🌌 Narrative Galaxy (Cluster Analysis)
A spatial mapping system for real-time news and narrative velocity. Allows institutional users to identify emerging themes and market shifts before they reflect in price.

### 🏊 Order Flow "GOMM" (Deep Dive)
A granular surveillance tool for Whale movements and temporal liquidity heatmaps. Features the institutional **Whale Watch** registry.

### 🧪 Strategy Lab (AI Backtesting)
A high-fidelity backtesting suite for AI-driven strategies. Test EMA crossovers and Alpha-weighted signals against historical data with slippage modeling.

### ⚗️ Portfolio Lab & Risk Engine (Phase 6)
A sophisticated simulation environment for Alpha-weighted funds.
- **Dynamic Rebalancing**: Fund updates constituent weights based on daily ML scores.
- **Risk Analytics**: Live tracking of VaR 95%, Portfolio Beta, and Sortino Ratios.
- **Correlation Engine**: 15x15 rolling 30-day asset dependency matrix.
- **Detailed Docs**: See [ANALYTICS.md](./ANALYTICS.md).
